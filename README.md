# PDX Pallet Supply

## PDX Pallet Supply - pallet supplier marketing site - [https://pdxpallets.vercel.app](https://pdxpallets.vercel.app/?utm_source=github&utm_medium=readme)

### Installation & control

```sh
npm i
```

### Dev-mode

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tech Stack

- React 19
- Next.js 16 (App Router) + SSR
- Tailwind CSS v4
- Lucide React
- TypeScript
